1 Install Java >=1.8 (If you haven't installed)

   1.1 Download Java 1.8 from https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html
   1.2 Install it.
   1.3 Check if Java works well (In terminal: run java -version)


2 Download AngClust files according to your OS
	
	Demo data
	demo_data.txt                   Demo gene expression data file.

	Windows
	AngClust_Readme_Win.txt         Readme of AngClust on Windows.
	AngClust_Tutorial_Windows.mp4   Video tutorial of how to use AngClust on Linux.      
	AngClust_Code_WinLinux.zip      Java code to implement AngClust in Eclipse on Windows and Linux.   
	AngClust_Win_x64.jar            Executable jar package of AngClust on 64-bit Windows.
	AngClust_Win_x86.jar            Executable jar package of AngClust on 32-bit Windows.

	Linux
	AngClust_Readme_Linux.txt       Readme of AngClust on Linux.  
	AngClust_Tutorial_Linux.mp4     Video tutorial of how to use AngClust on Linux.
	AngClust_Code_WinLinux.zip      Java code to implement AngClust in Eclipse on Windows and Linux.   
	AngClust_Linux_x64.jar          Executable jar package of AngClust on 64-bit Linux.
	AngClust_Linux_x86.jar          Executable jar package of AngClust on 32-bit Linux.

	MacOS
	AngClust_Readme_MacOS.txt       Readme of AngClust on Mac OS.  
	AngClust_Tutorial_MacOS.mp4     Video tutorial of how to use AngClust on Linux.    
	AngClust_Code_MacOS.zip         Java code to implement AngClust in Eclipse on Mac OS.
	AngClust_MacOS.jar              Executable jar package of AngClust on Mac OS.
			

3 Read Readme or watch tutorial video
	

4 Prepare your data file or use demo_data.txt
	PrepareData.xlsx can be a guideline.

5 Run AngClust
